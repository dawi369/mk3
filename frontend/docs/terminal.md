# Terminal

## Beta Goal

The terminal should provide a stable futures watch-and-chart experience for the supported contract universe.

## Included In Beta

- sector-organized terminal cards
- spotlight search
- ticker modal
- comparisons and spreads
- current session context from backend snapshots/sessions
- chart history over the retained backend window

## Explicitly Coming Soon

- AI Lab
- Backtesting

These can remain visible as placeholder surfaces, but they are not part of the functional beta core.

## Dev-Only Surface

- Stream view

The stream/debug view should not be part of the normal beta navigation.

## UI Rules For Beta

- No fake market prices
- No fake session statistics
- Missing or stale data must render as missing or stale
- Placeholder views must be visually obvious as coming soon
