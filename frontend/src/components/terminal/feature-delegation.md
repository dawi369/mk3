# View Delegation & Responsibilities

This document defines the core focus and feature set for each of the four primary views in the Terminal. This ensures a clean user experience without feature overlap.

## 1. Terminal (Market Radar)

**Core Focus**: Real-time market scanning and immediate price action.
**Responsibilities**:

- High-level overview of Gainer/Loser performance.
- Ticker-specific deep dives via modal.
- Raw price, volume, and intraday sparklines.
- **Goal**: High-speed awareness.

## 2. Indicators (Technical Authority)

**Core Focus**: Pure technical analysis, volatility, and trend regimes.
**Responsibilities**:

- "The Play": Authoritative daily/hourly trading strategies.
- Trend Strength & Directional Bias gauges.
- Volatility Regimes (Expanding/Compressing).
- Support/Resistance & Pivot levels.
- **Goal**: "What is the technical setup telling me to do?"

## 3. Sentiment (Market Psychology)

**Core Focus**: Human behavior, social context, and positioning.
**Responsibilities**:

- Order Flow Sentiment (Longs vs. Shorts).
- Social Sentiment (Crowd perception).
- Fear & Greed / Risk-On vs. Risk-Off.
- News-driven sentiment analysis.
- **Goal**: "Is the crowd right or wrong?"

## 4. AI Lab (Predictive Compute)

**Core Focus**: Machine learning signals and complex correlations.
**Responsibilities**:

- ML-driven price predictions.
- Dynamic correlation matrices between assets.
- Backtesting performance for specific strategies.
- Statistical anomaly detection.
- **Goal**: "Where is the hidden edge?"
